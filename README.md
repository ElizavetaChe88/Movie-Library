# Movie Library (Личная кинотека)

**Автор:** Чупрова Елизавета Игоревна

## Описание

Movie Library — это GUI-приложение на Python для хранения и управления личной коллекцией фильмов. Приложение позволяет добавлять фильмы, указывая их название, жанр, год выпуска и рейтинг, фильтровать данные по жанру и году, а также сохраняет все данные в JSON-файл для последующей загрузки.

### Функциональность

- Добавление фильмов с валидацией данных:
  - Год — целое число (1888–2030)
  - Рейтинг — число от 0 до 10
- Отображение фильмов в таблице
- Фильтрация:
  - По жанру (частичное совпадение, регистронезависимо)
  - По году (точное совпадение)
- Сохранение и загрузка данных через JSON
- Графический интерфейс на Tkinter

## Установка и запуск

1. Убедитесь, что установлен Python 3.6 или выше.
2. Скачайте файл `movie_library.py` и `movies.json` (создастся автоматически).
3. Запустите приложение:

```bash
python movie_library.py

## Примеры использования

Пример 1: Добавление фильма
В поле "Название" введите: Интерстеллар

"Жанр": Фантастика

"Год выпуска": 2014

"Рейтинг": 8.6

Нажмите "Добавить фильм"

Фильм появится в таблице.

Пример 2: Фильтрация по жанру
В поле "Фильтр по жанру" введите драма

Нажмите "Применить фильтр"

В таблице останутся только фильмы жанра "Драма" (регистр не важен)

Пример 3: Фильтрация по году
В поле "Фильтр по году" введите 1994

Нажмите "Применить фильтр"

Увидете все фильмы 1994 года

Пример 4: Ошибки валидации
Попробуйте ввести год abcd — появится ошибка "Год должен быть целым числом!"

Попробуйте рейтинг 15 — ошибка "Рейтинг должен быть от 0 до 10!"
