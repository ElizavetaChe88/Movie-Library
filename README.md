# Movie Library (Личная кинотека)

**Автор:** Чупрова Елизавета Игоревна

## Описание

Movie Library — это GUI-приложение на Python для хранения и управления личной коллекцией фильмов. Приложение позволяет добавлять фильмы, указывая их название, жанр, год выпуска и рейтинг, фильтровать данные по жанру и году, а также сохраняет все данные в JSON-файл для последующей загрузки.

### Функциональность

- Добавление фильмов с валидацией данных:
  - Год — целое число (1888–2030)
  - Рейтинг — число от 0 до 10
- Отображение фильмов в таблице
- Фильтрация:
  - По жанру (частичное совпадение, регистронезависимо)
  - По году (точное совпадение)
- Сохранение и загрузка данных через JSON
- Графический интерфейс на Tkinter

## 📚 Примеры использования

### Пример 1: Добавление первого фильма

**Сценарий:** Вы хотите добавить в коллекцию фильм "Начало" (Inception).

**Пошаговая инструкция:**

1. В поле **"Название"** введите: `Начало`
2. В поле **"Жанр"** введите: `Фантастика`
3. В поле **"Год выпуска"** введите: `2010`
4. В поле **"Рейтинг"** введите: `8.8`
5. Нажмите кнопку **"Добавить фильм"**

**Результат:**
- Появится сообщение: "Фильм 'Начало' успешно добавлен!"
- Фильм отобразится в таблице с ID = 5 (если уже были другие фильмы)
- Внизу страницы обновится статистика

---

### Пример 2: Поиск фильмов по жанру

**Сценарий:** Вы хотите найти все фильмы в жанре "Драма".

**Пошаговая инструкция:**

1. В блоке **"Фильтрация"** найдите поле **"По жанру"**
2. Введите: `Драма`
3. Нажмите кнопку **"Применить фильтр"**

**Результат:**
- В таблице останутся только фильмы с жанром "Драма"
- Например: "Побег из Шоушенка", "Зелёная книга" и т.д.
- Сработает **регистронезависимый поиск** (можно вводить "драма" или "ДРАМА")

**Совет:** Чтобы вернуть все фильмы, нажмите **"Сбросить фильтр"**.

---

### Пример 3: Фильтрация по году выпуска

**Сценарий:** Вы хотите найти все фильмы, выпущенные в 1994 году.

**Пошаговая инструкция:**

1. В блоке **"Фильтрация"** найдите поле **"По году"**
2. Введите: `1994`
3. Нажмите **"Применить фильтр"**

**Результат:**
- В таблице отобразятся фильмы только 1994 года
- Например: "Побег из Шоушенка", "Криминальное чтиво", "Форрест Гамп"

---

### Пример 4: Комбинированная фильтрация

**Сценарий:** Вы хотите найти фильмы 2014 года в жанре "Фантастика".

**Пошаговая инструкция:**

1. В поле **"По жанру"** введите: `Фантастика`
2. В поле **"По году"** введите: `2014`
3. Нажмите **"Применить фильтр"**

**Результат:**
- В таблице покажутся только фильмы, соответствующие **обоим критериям**
- Например: "Интерстеллар" (если он есть в коллекции)

---

### Пример 5: Обработка ошибок валидации

**Сценарий 5.1: Некорректный год**

1. Попробуйте добавить фильм с годом `abcd`
2. Нажмите **"Добавить фильм"**

**Результат:**
- Появится красное сообщение: **"Год должен быть числом от 1888 до 2030!"**
- Фильм не будет добавлен

**Сценарий 5.2: Рейтинг вне допустимого диапазона**

1. Попробуйте добавить фильм с рейтингом `15`
2. Нажмите **"Добавить фильм"**

**Результат:**
- Появится сообщение: **"Рейтинг должен быть числом от 0 до 10!"**
- Фильм не будет добавлен

**Сценарий 5.3: Пустые поля**

1. Оставьте поле "Название" пустым
2. Нажмите **"Добавить фильм"**

**Результат:**
- Сообщение: **"Все поля обязательны для заполнения!"**

---

### Пример 6: Удаление фильма

**Сценарий:** Вы хотите удалить фильм "Криминальное чтиво".

**Пошаговая инструкция:**

1. Найдите фильм "Криминальное чтиво" в таблице
2. Нажмите красную кнопку **"Удалить"** в последней колонке
3. Подтвердите удаление в диалоговом окне

**Результат:**
- Фильм исчезает из таблицы
- Обновляется статистика (количество фильмов уменьшается)
